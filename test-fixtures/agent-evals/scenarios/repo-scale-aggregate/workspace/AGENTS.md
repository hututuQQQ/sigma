# Repository statistics

For broad repository-wide statistics, include source and test files ending in
`.ts`, `.js`, `.py`, or `.rs` under `src/`, `tests/`, and `crates/`. Exclude
`.git/`, `vendor/`, and `generated/`. Do not follow symbolic links or junctions.

Unless the request says otherwise, “lines” means physical lines, including
blank lines. Report the scope and exclusions with the result. Statistics tasks
are read-only and must not create helper files in the repository.
